## 📖 Description

Ce projet est un **curriculum vitae interactif** développé dans le cadre du module **Développement Web** en **Licence 2 Informatique** à l'**Université Iba Der Thiam de Thiès**.

Il s'agit d'une page web moderne, responsive et entièrement interactive qui présente mes compétences, formations, expériences professionnelles et projets personnels.

---

## 🛠️ Technologies utilisées

| Technologie | Description |
| :--- | :--- |
| **HTML5** | Structure sémantique de la page |
| **CSS3** | Mise en page, animations et design responsive |
| **JavaScript (Vanilla JS)** | Interactivité et comportements dynamiques |
| **Font Awesome** | Icônes vectorielles |
| **Google Fonts** | Typographie moderne (Inter) |
| **html2pdf.js** | Génération de PDF |
| **QRCode.js** | Génération de QR Code |

---

## ✨ Fonctionnalités

| Fonctionnalité | Description |
| :--- | :--- |
| 🌙 **Thème clair/sombre** | Bascule avec sauvegarde dans `localStorage` |
| 📱 **Menu hamburger** | Navigation mobile adaptative |
| 📷 **Photo de profil** | Affichage avec effet au survol |
| 📊 **Barres de compétences** | Animation au scroll (`IntersectionObserver`) |
| 📋 **Copie des coordonnées** | Téléphone et email copiés dans le presse-papier |
| 📱 **QR Code** | Généré dynamiquement avec adaptation au thème |
| 📄 **Téléchargement PDF** | Export professionnel du CV |
| 📝 **Formulaire de contact** | Validation côté client (nom, email, message) |
| 🔙 **Bouton retour en haut** | Apparaît au défilement |
| 🎨 **Design responsive** | Adapté à tous les écrans |
